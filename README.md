# Memory Cards Game

A simple and fun memory matching game built to test and improve your concentration skills. Flip cards, find matching pairs, and try to complete the board in the fewest moves possible! Time runs anyway, so find all the pairs as soon as possible!

🎮 **Features**

🃏 Card flipping animation
🔍 Match pairs of identical cards
⏱️ Move timer
🔄 Restart game
🎉 Win detection

🛠️ **Technologies Used**

HTML
CSS
TypeScript
React

🧩 **How to Play**

1. Click on a card to flip it.
2. Click on a second card to find its match.
3. If the cards match, they stay flipped.
4. If not, they flip back after a short delay.
5. Continue until all pairs are matched.
